# Vespers

Daily devotional engine — scripture, reflection, prayer journaling.

## What is Vespers?

A minimal, beautiful app that surfaces a daily scripture reading with reflection prompts, prayer journaling, and reading plan tracking. Offline-first, no account required, no ads.

## Features (Planned)

- **Daily scripture** — curated verse with swipeable pages
- **Reflection prompts** — guided questions for deeper engagement
- **Prayer journal** — write and save prayers alongside scripture
- **Reading plans** — track progress through multi-day plans
- **Open plan API** — churches can create and share custom reading plans
- **Journal history** — review past reflections and prayers
- **Offline-first** — no internet or account required
- **Beautiful typography** — warm, twilight-themed design

## Tech Stack

- **Flutter** (cross-platform: iOS, Android, desktop, web)
- **Local storage** for offline-first data

## Getting Started

```bash
flutter pub get
flutter run
```
